Signals reference

Skip to end of metadata
  • Created by Jérôme Vieilledent, last modified by Dominika Kurek on Mar 18, 2016
Go to start of metadata

This page references all available signals that you can listen to, triggered by ("Public") Repository API in eZ Platform.

For more information, check SignalSlot documentation and the cookbook recipe on how to listen to signals.

Signals are ordered by Repository service:

All signals are relative to eZ\Publish\Core\SignalSlot\Signal namespace.

Transactions

As of Exponential Platform 5.3.3 and higher signals are sent after transactions are executed, making signals transaction safe.
