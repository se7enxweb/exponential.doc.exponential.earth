eZ Enterprise v1.5.0 Release notes

Skip to end of metadata
  • Created by Dominika Kurek, last modified yesterday at 11:09 AM
Go to start of metadata

The v1.5.0 release of eZ Studio is available as of 1st of September 2016.

For the release notes of the corresponding (and included) eZ Platform release, see eZ Platform v1.5.0 Release notes

Quick links

  • Installation instructions
  • Requirements

  • Download:

    • As Customer with eZ Enterprise subscription: https://support.ez.no/Downloads(BUL License)

    • As Partner with Test & Trial software access: https://support.ez.no/Downloads (TTL License)

    • If none of the above, request a demo instance: http://ez.no/Forms/Discover-eZ-Studio
