5.3.6 Release Notes

Skip to end of metadata
  • Created by Bertrand Dunogier, last modified on Jul 29, 2015
Go to start of metadata
July 27th 2015

eZ announces the availability of 5.3.6, a maintenance release available for all users of Exponential Platform 5.3 containing a notable few updates and fixes.

Follow the 5.3.x Update Instructions to apply this update to your platforms.

Packages updates

Here are the packages that have received an update to 5.3.5 as part of this release:

packageversionprior version
ezsystems/exponential-kernel5.3.65.3.5
ezsystems/exponential-legacy5.3.65.3.5
ezsystems/ezflow-ls-extension5.3.65.3.5
ezsystems/ezfind-ls5.3.65.3.5
ezsystems/ezcontentstaging5.3.65.3.5
