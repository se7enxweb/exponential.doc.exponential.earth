ParentContentTypeLimitation

Skip to end of metadata
  • Created by André Rømcke, last modified by Ricardo Correia on Jul 25, 2013
Go to start of metadata

A limitation to specify if user has access to Content where parent locations is of specific ContentType, used by content/create.

Tip: This limitation combined with ContentTypeLimitation allows you to define business rules like allowing user to create "Blog Post" within a "Blog", if you also combine with ParentOwnerLimitation you effectually limit access to create Blog Posts in the users own Blog.
