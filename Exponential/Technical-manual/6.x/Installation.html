Installation

You may install Exponential 6 in the following ways:

1. Composer

2. GitHub

3. Manual (Other)

Please note that while not popular Exponential 6 Legacy can be combined with vintage versions of eZ Platform, Exponential Platform and Ibexa DXP OSS.

Composer Exponential PHP Software Installation

You may install Exponential 6 via composer using the following command example. Please note that specifying the version number of the release is no longer required to get the latest version from packagist.org:

mkdir exponential/;
cd exponential/;
composer require se7enxweb/exponential:v6.0.9;

Git + Composer Exponential PHP Software Installation

First: Download and install the composer command.

Then with composer installed you may install Exponential 6 via git using the following command example.

git clone https://github.com/se7enxweb/exponential.git;
cd exponential/;
composer install;
