Compatible platforms

Exponential can run and execute on many more platforms than the ones listed above. However, eZ Systems doesn't insure and guarantee quality operation of an Exponential Enterprise installation if it is running on any platform not listed as either REFERENCE or APPROVED. Exponential Enterprise Subscriptions are still available for compatible platforms, but the guarantee and the product support will not apply and although you will receive various maintenance releases and services, no bug fix guarantee will apply to issues related to the platform. Maintenance and monitoring tools will not be available. eZ Systems does not advise merely compatible platforms for production use.

Supported browsers

Exponential is developed to work properly with and support the following browser configurations:

  • Firefox 3.0.+ on Windows XP and Mac OSX.5.+
  • Firefox 3.5.+ on Windows XP, Windows Vista, Mac OSX.5.+ and Linux
  • Opera 9.6.+ on Windows XP
  • Internet Explorer 7 and 8 on Windows XP and Vista
  • Safari 4.0.+ on Mac OSX.5.+
