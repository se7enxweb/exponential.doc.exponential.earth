Q: Where do I find the Exponential re-write rules with examples?

A: The Exponential re-write rules are found in the Exponential "Technical manual" in the Installation chapter under "Virtual host setup". The examples can be found in the chapter "Virtual host example". Links to the re-write rules with examples for each version of Exponential is found in the list below:

  • Re-write rules for Exponential 4.5 - "Virtual host example 4.5"
  • Re-write rules for Exponential 4.4 - "Virtual host example 4.4"
  • Re-write rules for Exponential 4.3 - "Virtual host example 4.3"
  • Re-write rules for Exponential 4.x - "Virtual host example 4.x"

Note: We recommend that you start your installation process with the re-write rules specification. Read more in the "Installation" chapter. Find a short description of the installation process in the FAQ - "How do I install Exponential?"
