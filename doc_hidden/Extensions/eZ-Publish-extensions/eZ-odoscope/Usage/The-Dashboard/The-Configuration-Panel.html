The Configuration Panel

The Configuration Panel allows you to set up the Dashboard layout, export and import reports, and set a usergroup for all reports. To access the Dashboard Configuration Panel, click on the arrow shown on the top of its area:

eZ odoscope Dashboard Access

The Dashboard Configuration Panel area will expand and you will be able perform the actions described below:

eZ odoscope Dashboard Options

  • Edit the Dashboard Template: The four squares on the top of the blue area allow you to choose how many reports you want to show per row. You can choose by clicking on the desired template.        
  • Export/Import Options:
    • Export All to PDF: This option will export ALL your reports to a PDF file;
    • Export Selected do PDF: In order to use this option, you have to select a report (or reports) first. The selected report(s) will be exported to a PDF file;
    • Export Report to XML: This option will export the selected report(s) to an XML file;
    • Import Report from XML: This option will import a report (or reports) from an XML file and add it to the Cockpit.
  • Select Usergroup: This option allows you to select the same Usergroup to all your reports at the same time. Select the usergroup and click on "Set Usergroup for All Slides" button, and all your reports show the new data according to the usergroup you chose.
