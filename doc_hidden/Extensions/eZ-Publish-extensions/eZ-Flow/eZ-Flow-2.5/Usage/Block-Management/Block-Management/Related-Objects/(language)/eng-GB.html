Related Objects

The related objects view feature allows you to display the latest related content at the bottom of any published content. Any type of content can be listed as related content, varying from articles, to images and blog posts. They are selected based on the keywords they have in common with the viewed content. An example is given here.

In order to show this related content, tags must first be given to content using the keyword attribute. To do this, log in to the Administration Interface or the Front-end editing interface. Keep in mind that submitting tags is only possible when the keyword attribute is added to the content class you are using.

Once the correct tags have been added, the main view of each tagged content will be accompanied by a list of related content.
