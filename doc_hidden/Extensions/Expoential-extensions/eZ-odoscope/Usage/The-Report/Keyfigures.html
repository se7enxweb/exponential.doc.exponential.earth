Keyfigures

Keyfigures are performance indicators used in website analysis. They are closely tied to business processes.

On eZ odoscope Extension, a keyfigure is set upon one filter. The keyfigure is used to calculate the correlation of all filters regarding a specific filter (the filter used to create the keyfigure).

The "Bouncer" is eZ odoscope Extension default keyfigure. It is automatically created when the Extension is installed, and represents the Bounce Rate of your website.

Remember: The bounce rate represents the percentage of visitors who entered the site and "bounced" (left the site) rather than continue viewing other pages of the site.

When you create a new report, the keyfigure "Bouncer" will be automatically assigned to it.

Creating a Keyfigure

The Keyfigure, along with the three filtering levels: the Time Period (see The Calendar), the Usergroup and the Filtergroup allow you to analyze the data that is important for you to understand and analyze how the visitors behave in your website and take measures to improve its usage. The keyfigure allows you to have a quick view on your site's performance, and that is why it is so important.

Before you create a keyfigure, you have to select the filter you want to use for it. 

eZ odoscope Keyfigre Filter

After you have selected a filter (in the example above, the filtergroup is "Session Length" and the filter is "1", which represents the visitors who entered the website and left it without interacting with it), you can create a keyfigure for it, by right clicking on it and clicking on the option "Create keyfigure":

eZ odoscope Keyfigure Creation

The following window will be displayed:

eZ odoscope Keyfigure Indicators

This is the window where you configure the keyfigure indicators. Let's look at the information available on this screen.

Label: The label is the name of the keyfigure. By default, it is named after the filter you selected, but you can change it;

Active: If checked (Active), it means that the percentage shown for the keyfigure will be the percentage of visitors the filter applies to. If it is not checked (Inactive) it means that the percentage shown for the keyfigure will be the percentage of visitors the filter does not applies to (all the other filters).

 Keyfigure is Inactive 

eZ odoscope Keyfigure Inactive

How to interpret, i.e., Europe's values: 22.252 Visitors coming from Europe have a correlation value to your created keyfigure:

  •  Active set meaning: 61,13% of them had 1 Interaction with the website
  • Inactive set meaning: 38,87% of them had more than 1 one interaction (all the other filters, excepted the selected one)

The two values (Active and Inactive) shown for each set complement each other, and their total sum is 100%, i.e.: If you sum Europe's values - Active = 61.13 and Inactive = 38.87, the total is 100%.

Indicators Percentage: Inform the percentages you want to use for the keyfigure indicators (green, yellow and red). Remember that the keyfigure is set for ONE filter. Have in mind that the indicators percentage relates to the filter you have chosen. It may be subjective and can vary according to the business, but it has to have some sense.

Green/Red: Sets the indicator for the keyfigure to either green/red or red/green. Click on the button in order to set the indicator you want to use.

Example 1: Session Length, filter "1", which means users who left the website after just one interaction.

The less visitors had just one interaction, the better, so you may want to set the range between the green/yellow/red as follows:

eZ odoscope Keyfigure Bad Indicator

This is a BAD indicator, so you may want to set it from green to red and set the initial interval (green) really low. Remember: in this case, the less, the better.

Example 2: Session Length, filter "21-30", which means users who left the website after 21-30 interactions.

The more visitors have 21-30 interactions, the better, so you may want to set the range between the green/yellow/red indicators as follows:

eZ odoscope Keyfigure Good Indicator

This is a GOOD indicator, so you may want to set it from red to green and set the initial interval (red) not so low. Remember: in this case, the more, the better.

The keyfigures created will be shown in the middle of the odoscope Explorer area, as can be seen below:

eZ odoscope Keyfigures Created

Once you have created a keyfigure, you can use it for other usergroups and filtergroups, and you don't need to filter anymore.

The correlation does not change, but only the basis, when you add or remove the filters, as can be seen below:

eZ odoscope Keyfigure Correlations Example I

The basis are the values and percentages that are inside the green square. The correlations are the percentages that are inside the red square.

As can be seen, the correlations (the keyfigure applied to the filters chosen) don't change:

eZ odoscope Keyfigure Correlations Example II

You can perform the following actions regarding keyfigures: Save, Rename and Delete, respectively shown below, from left to right:

eZ odoscope Keyfigures Options

Once you have created your most used keyfigures, you can select them by clicking on them.

Remember: all your charts/reports will automatically change when you select a new keyfigure.
