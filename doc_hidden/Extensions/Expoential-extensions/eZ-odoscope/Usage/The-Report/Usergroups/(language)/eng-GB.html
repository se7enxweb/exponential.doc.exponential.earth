Usergroups

With the help of the odoscope Explorer and the Ad Hoc Filtering, you can create groups of segments quickly and intuitively.

If there is a segment you need to analyze often, you can save it as a usergroup.

The "All Users" usergroup is odoscope's default usergroup. It is automatically created when eZ odoscope is installed, and is assigned to every new report you create. As its name says, it represents all users that have visited your website in a given period of time.

Creating a Usergroup

One of the filters of the report is the usergroup, along with: the Time Period (see the Calendar) and the Filtergroup.

To create a usergroup, you need to select the filters you want (or dont' want) to apply to it.

Ad Hoc Filtering

The Ad Hoc Filtering is an odoscope functionality that allows you to select any filter, at any level, and see the results immediately. Please, refer to the Ad Hoc Filtering documentation in order to be familiar with it.

eZ odoscope Usergroups

If there is a combination of filters you use often, you can save it as a Usergroup.

1) Right-click on the odoscope Explorer area and select the option "Create usergroup":

eZ odoscope Creating a Usergroup

2) The new usergroup will be shown highlighted in the usergroups area:

eZ odoscope Usergroup Name

3) Rename it to a name of your choice.

You can perform the following actions regarding usergroups: Save, Rename and Delete, respectively shown below, from left to right:

eZ odoscope Usergroup Options

All usergroups created are shown in the odoscope Explorer, in the usergroups area.

eZ odoscope Usergroups

Once you have created a usergroup, you can select it by clicking on it.

Remember: all your charts/reports will automatically change when you select a new usergroup.
