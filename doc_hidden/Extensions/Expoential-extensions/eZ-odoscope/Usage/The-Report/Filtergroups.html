Filtergroups

The Filtergroups are predefined within the eZ odoscope Extension.

Note: The number of Filtergroups is not limited, and can be extended. Please, consult eZ Systems in order to know how to extend them.

The Filtergropus represent the most common variants used in web analysis, such as:

eZ odoscope Filtergroups

All Filtergroups refer to visitors' (i.e.: Pages Accessed by the visitors, Operating System the visitors used to access your website, where - Origin - are your visitors from, and so on), and they give you a clear insight about visitors behavior on your web site, between other important information. Combined with Usergroups, Keyfigures and the Time Period (see the Calendar), they are a powerful tool for web site analysis.

All Filtergroups are in correlation between each other, i.e.: A visit has exact one value in every filtergroup. ‘Not set’ is also a value – meaning that this visit has no data entry in this filtergroup. That said, the main filtergroups must be all checked , so you are able to have data displayed on your charts/reports.Then you can select/deselect the filters (not the filtergroups) you want to use use on them.

When you create a new Report (or at any time you want to change the information on a report/chart), you must select a main filtergroup (the main filtergroup is the filtergroup which data you want to analyze on your chart/report, i.e.: Session Time, Day of The Week, Browser, Origin, etc). You can do this by right clicking on your report's chart, then clicking on "Data" and finally clicking on "Filtergroups". You will see a list of the Filtergroups available and then you will be able to select the main filtergroup. Please, refer to the Report documentation, "Selecting a Filtergroup", in order to be familiar with this procedure.

 You can select the filters for your chart/report with the odoscope Explorer. Let's see how to do this.

A Filtergroup has several filters, and can have many levels (hierarchies). A good example is the 'Origin' filtergroup:

eZ odoscope Filtergroups Hierarchy

As you can see, there are the following levels for Origin:

  • Region (Continent)
  • Country
  • County (Province, State)
  • City

Each level is a filter itself. In the example above, you can see that the final level for the Origin filtergroup is the filter City. This means you can filter from which city your visitors' come from.

You can also see some very helpful information regarding filters, in the odoscope Explorer area: 

eZ odoscope Filtergroups Info

Let's see what each one means:

  • Label: The label is the name of the filter (or Filtergroup). You cannot modify this information;
  • Visits: On this column you will see the relative number of visits that matches the filter for the level you are looking at, in the time period selected;
  • %: On this column you will see the percentage of visits calculated for a filter, regarding the overall amount of visits for all filters of the same level;

          Example: Imagine you have a total number of 1000 visits from Germany, and a total number of 9000 visits from all other European countries. You should see 10%, on this column, for Germany, as the total number of visits from Europe is 10000, and 1000 is 10% of 10000.

  • 'Bouncer' (%): On this column you will see the percentage of the keyfigure you are using. The keyfigure is explained in a separate documentation page. Refer to the following link in order to be familiar with it: Keyfigures.

You can go deeper in the levels and select/deselect the filters you want to apply to your chart(s)/report(s).

You can perform the following actions, regarding Filtergroups and filters:

  • Expand: Click on the arrow shown at the left of the Filtergroup/filter name to expand it. When you do this, you will be able to see the next level available; 
  • Select/Deselect: Click on Filtergroup/filter to select/deselect it. All under the filter you have selected will be selected/deselected;
  • Select a Unique Filter: You can select a unique filter by SHIFT-clicking on the filter you want to select.This will select ONLY the filter you Shift-clicked on, and deselect all the other filters in the same level. Note: This action cannot be performed at the Filgregroups level;
  • Select Multiple Filters: You can select multiple Filtergroups/filters by clicking on the Filtergroup(s)/filter(s) you want to select.  

         Note: You have to deselect all the filters, by clicking on their superior level, so you can select only the filters you want. i.e.: If you want to select only the countries Germany, UK and France, within the filter Europe, you have to do the following:

  • Expand the Origin Filtergroup;
  • Deselect the filter Europe;
  • Click on the filters: Germany, United Kingdom and France.

 Let's take a look at the eZ odoscope Explorer Signaling:

eZ odoscope Explorer Signaling

  • Empty Square: The filter is not selected;
  • Square with a Trace: The filter is partially selected;
  • Checked Square: The filter is totally selected.

Hint: If you look at the filter "Ile-the-France", you will see the following:

eZ odoscope Explorer Filter Page Navigagion

This means that the filter "Ile-de-France" has seven pages, and you are looking at its first page content. You can navigate on them by clicking on the left and right arrows shown together with the page numbers.

Remember: Every time you select/deselect a filter, your charts/reports automatically change, according to the data available for the time period chosen in the Calendar.

You can combine filters (Ad Hoc Filtering) you use frequently and create Usergroups. Please, refer to the previous links in order to be familiar with both functions.
