The Dashboard

The Dashboard

The Dashboard is the area where you create and see your reports, the area where all the recorded data on your website is displayed graphically.

Note: This is eZ odoscope first level, in terms of views. The second level is the Report view, and you can access it by double clicking on a report (zoom in).

eZ odoscope Dashboard

There are some important functions you can access and several actions you can perform within the Dashboard area. Let's take a look at them.

Reports

All your reports are created and shown within the Dashboard Area.

You can perform the following basic actions regarding Reports, on this screen:

  • Add a new report
  • Delete an existing report
  • Modify an existing report
  • Rename an existing report
  • Reorganize your reports' slides (move them within the Dashboard area)
  • Select a filtergroup for your report. See the Filtergroups documentation, please, for more details on this functionality.
  • Select a usergroup for all your reports. See the Dashboard Configuration Panel documentation, please, for more details on this function.

Please, refer to the Report documentation in order to get familiar with these and more advanced options.

Selecting a Time Period

The time period is the first level of filtering on eZ odoscope. The most important basis of your report is the period of time you want to analyze. To select the Time Period, you must open the Calendar and set the dates interval for your report(s).

Note: By default the time period spans the 30 last days .

Selecting a Usergroup

All slides are created with the default usergroup "All Users" but you can change it to another usergroup of your choice.

You can select a usergroup for your report by clicking on the right upper corner of the gray square around the charts, where it says "All Users".

When you click on this area, you will be able to see all the Usergroups you have created and then choose which one you want to use on your report.

Note: Refer to the Usergroups documentation in order to be familiar with Usergroups meaning, creation and management.

Setting up the Chart

To access this option, right click where it says "Right click to setup the chart". The following screen will be shown:

eZ odoscope Setting Up Chart

You can perform the following actions, on this screen:

  • Add a new slide to your report: A blank slide will be added to your report;
  • Copy an existing slide of your report: A slide identical to the slide being copied will be added to your report;
  • Export PDF: A PDF format file will be created with the report image;
  • Export Image: A PNG format file will be created with the report image.

Hint: To exit this page, left click on it.

The Dashboard Configuration Panel

At the left side of the Dashboard, you can see the Dashboard Configuration Panel. You can expand it by clicking on the little white arrow available at the right top corner of its area:

eZ odoscope Dashboard Configuration Panel

The Dashboard Configuration Panel allows you to perform several actions, and one of them is to change the number of slides shown per line (the default number of slides per line is three). Please, refer to the Dashboard Configuration Panel documentation in order to be familiar with it.

Organizing Your Reports

You can organize your reports within the Dashboard area. Just click (just one click) on them and move them around till you find which displaying fits best for you.
