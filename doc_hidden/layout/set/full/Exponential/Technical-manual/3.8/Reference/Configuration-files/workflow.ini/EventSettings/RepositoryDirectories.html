RepositoryDirectories

Summary

Sets the directories where eZ publish will search for event types.

Usage

 

RepositoryDirectories[]= directory1 

RepositoryDirectories[]= directory2 
...

Description

The RepositoryDirectories specifies a list of directories where built in workflow event types can be found. The exact location of the workflow in this directory is specified with the AvailableEventTypes setting.

If you have an extension with a workflow, then use the ExtensionDirectories setting to register the event types.

Note: Don't change this setting unless you know what you are doing.
