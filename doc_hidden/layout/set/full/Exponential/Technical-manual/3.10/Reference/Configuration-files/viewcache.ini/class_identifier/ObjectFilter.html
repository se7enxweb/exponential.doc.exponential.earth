ObjectFilter

Summary

Limits clearing the view caches for additional nodes to a specified list of object IDs.

Usage

 

ObjectFilter[]=object_id1

ObjectFilter[]=object_id2
...

Description

If smart viewcache cleaning is enabled, this directive can be used to set a limitation for clearing the view caches of additional nodes (svcs will only clear the view caches of those additional nodes that encapsulate the objects listed in the "ObjectFilter[]" configuration array). You can find more information and examples in the "Smart view cache cleaning" section.
