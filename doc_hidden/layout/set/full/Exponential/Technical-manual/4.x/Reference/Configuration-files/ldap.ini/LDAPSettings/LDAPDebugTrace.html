LDAPDebugTrace

Summary

This setting enables tracing the LDAP login.

Usage

 
LDAPDebugTrace= disabled | enabled

Description

This setting enables tracing the LDAP login, which outputs extensive debug information which can be used during setup.
Keep in mind that login name and passwords will be logged to logfiles or outputted if DebugOutput settings are enabled in site.ini. For this reason it is best not keep this setting enabled on production setup.
This setting is case sensitive.

Examples

LDAPDebugTrace=disabled

No debug information will be shown during setup.
