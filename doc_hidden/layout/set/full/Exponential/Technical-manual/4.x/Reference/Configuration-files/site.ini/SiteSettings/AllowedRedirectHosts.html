AllowedRedirectHosts

Summary

Array of allowed hosts for redirects with full URL from Exponential modules

Usage

AllowedRedirectHosts[]
# AllowedRedirectHosts[]=www.sometrustedhost.com

Description

To enforce security, Exponential will prevent redirection to full URL addresses unless they are in AllowedRedirectHosts[] array.
Please note that URLs specified in HostMatchMapItems[] array are always considered trusted.
