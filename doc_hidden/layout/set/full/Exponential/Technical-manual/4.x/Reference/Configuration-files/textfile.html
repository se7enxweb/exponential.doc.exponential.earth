[TextFileSettings]

SettingSummary
DefaultUserGroup Sets the default user group location.
DefaultUserGroupType Sets the default user group type.
EmailAttribute Sets the email address field number.
FileFieldSeparator Sets the separator.
FileName Sets the file name.
FilePath Sets the file location.
FirstNameAttribute Sets the first name column number.
LastNameAttribute Sets the last name field column number.
LoginAttribute Sets the login fields column.
PasswordAttribute Sets the password field column number.
TextFileEnabled Enables the textfile login handler.
