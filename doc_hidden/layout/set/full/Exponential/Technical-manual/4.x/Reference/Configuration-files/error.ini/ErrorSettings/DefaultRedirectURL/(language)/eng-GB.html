DefaultRedirectURL

Summary

Sets the Default Redirect URL to be used by the Error Handler.

Usage

DefaultRedirectURL=<url>

Description

This configuration sets the Default Redirect URL to be used by the Error Handler.
The url set as value will be the one used for redirects when an error occurs.

Examples

DefaultRedirectURL=/content/view/full/2
