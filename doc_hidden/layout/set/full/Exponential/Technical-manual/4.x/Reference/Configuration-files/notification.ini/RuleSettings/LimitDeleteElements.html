LimitDeleteElements

Summary

Limits the number of elements to be deleted when sending notifications in digest mode.

Usage

 
LimitDeleteElements =<integer>

Description

This sets the maximum number of notification digest elements that are to be removed in one query. This is needed to prevent exceeding the DBMS limits (maximum query length, maximum number of elements).

Examples

LimitDeleteElements=50

This is the default setting.
