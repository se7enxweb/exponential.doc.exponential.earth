IncludeList

Summary

Lists the PHP files that should be included when the "simpletags" operator is used.

Usage

IncludeList[]=<include name>

Description

This setting lists the PHP files that should be included when the "simpletags" operator is used.It will look in the kernel/classes/simpletags folder or the simpletags subdirectory of the extensions defined in the setting [SimpleTagsOperator] Extensions.

This setting is case sensitive.

This setting is left empty by default.

Examples

IncludeList[]=
