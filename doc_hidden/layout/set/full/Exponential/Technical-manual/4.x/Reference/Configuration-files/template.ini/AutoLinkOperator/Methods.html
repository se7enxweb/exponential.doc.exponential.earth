Methods

Summary

This setting lists the URL methods or protocols that should be matched.

Usage

Methods[]=<methode_name>

Description

This setting lists the URL methods or protocols that should be matched.

This setting is case sensitive.

Examples

Methods[]=http
Methods[]=https
Methods[]=ftp
Methods[]=ftps
Methods[]=webdav
Methods[]=webdavs

These are the default settings.
