femto

Summary

Sets the base unit for femto.

Usage

 
femto=<factor>;<symbol>

Description

This setting sets the factor value and the symbol for an array to use for femto. The factor is used for calculating values in the power of 10.
This setting is case sensitive.

Examples

femto=-15;f

This is the default setting, indicating that the base unit for femto is "f" and the factor is 10 -15.
