DefaultIcon

Summary

Sets the default icon to use.

Usage

DefaultIcon=<icon_name>

Description

This setting sets the default icon to use for flag icons.  This setting is case sensitive.

Examples

#DefaultIcon=unknown

By default this setting is set to unknown and commented out.
