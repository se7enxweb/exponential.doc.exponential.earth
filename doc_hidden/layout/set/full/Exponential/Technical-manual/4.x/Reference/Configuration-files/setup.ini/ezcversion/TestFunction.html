TestFunction

Summary

Testing the eZ Components function during setup.

Usage

 
TestFunction=<function>

Description

This setting tests if the eZ Components version installed is greater than required version, by testing the function.
The value "walkrecursive" is the default value, for more information regarding "walkrecursive" please visit http://ezcomponents.org/docs/api/latest/Base/ezcBaseFile.html#walkRecursive

This setting is case sensitive.

Examples

TestFunction=walkrecursive

This is the default setting.
