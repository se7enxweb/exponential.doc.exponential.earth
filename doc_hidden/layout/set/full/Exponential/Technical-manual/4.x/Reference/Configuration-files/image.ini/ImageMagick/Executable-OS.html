Executable{OS}

Summary

Sets the executable that must be used for a specific operating system.

Usage

ExecutableWin32=convertim | convert
ExecutableMac=convertim | convert
ExecutableUnix=convertim | convert

Description

“ExecutableWin32” sets the name of the executable for Windows. “ExecutableMac” sets the name of the executable for the Mac specific converter. “ExecutableUnix” sets the name of the executable for the Unix/Linux specific converter.

Possible values:

  • convert
  • convertim

This setting is case sensitive.

Examples

ExecutableWin32=convertim
 
ExecutableMac=convert
 
ExecutableUnix=convert
