DBPassword

Summary

This sets which password to use for your database when clustering with the eZDB File Handler.

Usage

 
DBPassword=<password>

Description

This sets which password to use for your database when clustering with the eZDB File Handler. Use the actual password as value, but you can also leave the password field empty.

In most cases this value will be the same as the "Password" setting specified under the [DatabaseSettings] block of your "site.ini.append.php" configuration file.
When using an eZ DFS File Handler please refer to the settings in file.ini [eZDFSClusteringSettings].

This setting is case sensitive.

Examples

DBPassword=mypassword
