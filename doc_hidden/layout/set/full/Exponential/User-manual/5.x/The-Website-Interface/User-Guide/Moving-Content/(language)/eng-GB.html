Moving Content

To move content from one location to another on the site, use the Move button on the Website Toolbar. After clicking on Move, you will be able to browse the web site and choose the new location for the content.

You can either choose the new container by clicking on the radio button next to the name or click the container name to view the contents of the container.

After selecting the new location, click on the OK button.

Technical info

This functionality is using the content/move view.

Note: If you move a content container the system will also move any content beneath that object. For example, if you move a folder containing articles from one location to another, the articles will also be moved.

Note2: Moving content currently depends on the user having access to content/move permission, in addition to having access to remove the content in question at the specified location and likewise access to create this content in the target location.
