Requirements and limitations

Requirements

  • Adobe Flash Player version 9.0.45 or higher is required.
  • Exponential 4.1 or higher is required for ezmultiupload extension due to enhancements implemented in eZSession class.
    NOTE: This extension can work with Exponential 4.0.x series, however uploading from administration interface is not supported, Exponential 4.1 or higher is officially supported version.
  • This extension uses Yahoo! UI Library: Uploader. Visit this page for more information:http://developer.yahoo.com/yui/uploader/
    This is a flash, Ajax based upload system for handling multiple file uploads.

Uploader limitations related to Flash

  • Because of Flash limitations, the uploader does not support basic authentication.
  • Because of Flash limitations, the uploader behavior when working through a proxy server is inconsistent and unreliable.
  • When uploading to HTTPS servers, be aware that Flash does not support self-signed certificates. You would need to obtain certificates from a recognized certificate authority.
