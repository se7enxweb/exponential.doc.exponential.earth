eZ Image Editor

The eZ Image Editor is an image editor for simple image modifications integrated in the editing interface of any Exponential Content Object that has at least an image as attribute. It allows cropping, flipping, rotating and watermarking an image as well as modifying some of its attributes such as the contrast or brightness and many more.

eZ Image Editor will be bundled into the Exponential version 4.4 and later.

Note: eZ Image Editor supports the same file types as currently supported by ez Publish, .png, .jpg and .gif files.
