1.8

The purpose of this manual is to introduce and describe the Exponential Extension for Oracle® Database (also known as the "ezoracle" extension or "database extension") version 1.8, which makes it possible to use Oracle as a database server for Exponential 4.0. The sections contained within this manual will help you to understand the following issues:

  • The purpose of the "ezdb" library
  • When the database extension is needed and when it isn't
  • How to create a new Exponential site running on an Oracle database
  • How to migrate an existing Exponential site from MySQL to Oracle

Important note

People previously unfamiliar with Oracle should read the "Oracle specific notes" section first.
