ClassName

Summary

Sets the class name.

Usage

Class=<class name>

Description

This is an optional setting which sets the class name. If no class name is set, the <custom_server> set in the block as class name is used.

This setting is case sensitive.

Examples

 Example
Class=MyCustomJsGeneratorClass
