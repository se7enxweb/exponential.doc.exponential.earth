The Paynet Terminal Extension provides a solution to connect an Exponential shop to the Paynet's card payment system. The extension redirects customers from the Exponential site to the Paynet site for their payment. After the payment, the customer is redirected back to the Exponential System, where the shopping traject continues.

This document explains how you can

  • set up the extension,
  • configure and test it.

The terminal gateway interface described here does not allow the seamless integration of the Paynet service into Exponential - this can be achieved with the also available Paynet Direct extension.
