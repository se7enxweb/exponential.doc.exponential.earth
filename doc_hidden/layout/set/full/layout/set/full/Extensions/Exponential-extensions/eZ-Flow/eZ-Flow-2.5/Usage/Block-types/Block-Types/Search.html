Search

A simple search block can be added to search your Exponential content. Log in to the eZ Flow front-end editing interface to edit the front page. Select a zone and under Block select Search and click "add block".

Give the search block a name in the screen below.

Click "Send for publishing" to complete the procedure. This block provides a simple search form based on the search box used on the front page and will look like this:

Search results are listed using an Ajax call without reloading the page.
