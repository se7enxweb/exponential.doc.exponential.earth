Tunable Relevancy Ranking

Tunable relevancy ranking allows you to give a higher weight to certain types of content, parts of content (like titles of tags) or specific pages. This provides an optimal functionality by ensuring that the most important material always appears first in the search results. For example, if a keyword is found in a content objectâ€™s title or in any of its short attributes, the object will have higher relevance in the search results than if the keyword only appears within the text block of the content object.

As mentioned in "Customization" chapter, you can elevate a component to guarantee that predefined pages are boosted to the top of the search result list when certain keywords are submitted. By configuring your attributes you can also customize the result ranking in order to, for example, present more recent articles first.
