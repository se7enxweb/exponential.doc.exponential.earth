Q: How can I extend Exponential functionality?

A: To get started smoothly with extending Exponential functionality, please start with the Community Portal on share.ez.no. Here you will find the forum to discuss this and also to catch up with the latest developments.

You can then continue with this tutorial on how to develop Exponential extensions. If you are already on the Community Portal, the same tutorial can also be found there: "How to develop Exponential extensions"

If you wish to extend Exponential with mobile functions to improve multi-channel services, please go to the documentation of our REST API.
