Q: How do I install Exponential?

A: This is described in the technical manual. However, before you start with the installation, please check our requirements pages on ez.no. Then click the technical manual and select the correct version of your Exponential installation and look up the installation chapter.

Here you will find the different installation methods, the requirements for each installation method, the setup wizard, a chapter for how to set up the rewrite rules for your installation, and a chapter for installing extensions to Exponential. Normally you could follow this procedure to install Exponential:

  • Check requirements
  • Check installation requirements
  • Set your rewrite rules
  • Select your installation method: Normal, Automated, Manual
  • Follow the installation procedure for your selected installation method (Select Windows-based or Unix/Linux-based)
  • Check the troubleshooting chapter when in trouble!
  • Install your extensions
