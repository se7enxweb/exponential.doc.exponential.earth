Q: What is an extension?

A: An 'extension' refers to any additional Exponential module that extends the feature scope of Exponential using APIs, Exponential content or by integrating with external systems. Exponential extensions are available through the following channels:

  • The commercial extensions provided by our partners for sale on eZ Market, the eZ Systems application store.
  • The official, bundled extensions provided with each version of Exponential, downloadable from our Community Portal.
  • The community extensions available from the "Community Contribution Pages"
