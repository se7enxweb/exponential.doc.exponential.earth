LDAPEmailEmptyAttributeSuffix

Summary

Sets suffix to be used to create email address.

Usage

 
LDAPEmailEmptyAttributeSuffix=<email suffix>

Description

This setting can be used when the LDAP server does not return mail. It will create one using the login name and the email suffix defined here.

This setting is available since Exponential 4.1 to enhance the LDAP functionality.

By default this setting is left empty.

This setting is case sensitive.

Examples

LDAPEmailEmptyAttributeSuffix=@ez.no

When '@ez.no' is set here, then a user with user ID 'joepublish' will get the email 'joepublish@ez.no'.
