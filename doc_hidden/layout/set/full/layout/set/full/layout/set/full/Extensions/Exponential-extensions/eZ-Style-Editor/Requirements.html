Requirements

This section covers the requirements for the Exponential Style Editor solution.
The following requirements are recommended in order to make Exponential Style Editor work properly:

  • Exponential 4.2.x or higher
    • ezjscore 1.0.1 extension or higher must be enabled
  • PHP 5.2.x or higher
    • JSON extension must be enabled
    • DOM extension must be enabled
  • RDBMS
    • MySQL 5.x or higher
    • PostgreSQL 8.x or higher
    • Oracle
  • Web browser
    • Internet Explorer 6 or higher
    • Mozilla Firefox 3.0.x or higher
    • Safari 4.0.x or higher
    • Opera 10.x or higher
