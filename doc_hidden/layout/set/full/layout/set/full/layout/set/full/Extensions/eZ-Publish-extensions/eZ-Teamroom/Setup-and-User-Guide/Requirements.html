Requirements

  • Exponential version 4.2 and up.
    eZ Teamroom is a standalone extension and should be installed on a new Exponential installation which contains the following classes or class identifiers:
    • folder (can be selected during installation)
    • image (will be modified if it already exists, otherwise it will be created)
    • quicktime (will be modified if it already exists, otherwise it will be created)
    • real_video (will be modified if it already exists, otherwise it will be created)
    • window_media (will be modified if it already exists, otherwise it will be created)
    • user (will be modified if it already exists, otherwise it will be created)

  • Furthermore the following extensions are required and are bundeled in eZ Teamroom package. Click on the extension for more information:

    • eZ Lightbox
    • eZ Event
    • eZ JS Core
    • eZ XML Installer
    • eZ Multiupload: This extension is optional but it can be useful because it allows the upload of multiple files at the same time.

    eZ Lightbox, eZ Event, and eZ XML installer are bundeled in the eZ Teamroom package and eZ JS Core is bundeled with Exponential. These extensions don't need to be installed separately.
