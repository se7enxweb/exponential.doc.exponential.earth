Item list

This block allows you to show any type of content on your front page in the form of a list of titles. The following screenshot shows an example for blog contents.

To add an Item list to your front page, log in to the eZ Flow front-end editing interface to edit the front page. Select a zone and choose "Item list" in the block drop-down-list.

The following block will be added to the block list, where you are able to name the block. Next add items and define the additional parameters such as overflow and the rotation frequency of the content, set in minutes, hours or days.

And click "Send for publishing" to complete the procedure.
