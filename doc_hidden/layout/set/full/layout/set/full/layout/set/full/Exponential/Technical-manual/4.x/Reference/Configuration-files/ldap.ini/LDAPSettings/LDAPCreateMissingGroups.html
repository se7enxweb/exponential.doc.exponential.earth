LDAPCreateMissingGroups

Summary

Sets whether or not groups that are not found in Exponential should be created.

Usage

 
LDAPCreateMissingGroups= disabled | enabled

Description

When the setting LDAPGroupMappingType in block [LDAPSettings] is set to value UseGroupAttribute, LDAPCreateMissingGroups sets whether or not groups that are not found in Exponential, should be created.
Enabling it will allow groups to be created, disabling is will prevent groups to be greated.

This setting is case sensitive.

Examples

LDAPCreateMissingGroups=disabled

None existing groups will not automatically be created.
This is the default setting.
