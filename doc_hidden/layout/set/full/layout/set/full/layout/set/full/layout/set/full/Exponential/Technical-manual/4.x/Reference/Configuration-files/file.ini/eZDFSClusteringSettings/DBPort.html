DBPort

Summary

This setting sets which port to use when clustering with the eZDB File Handle

Usage

 
DBPort=<port number>

Description

This setting sets which port to use when clustering with the eZDB File Handler.

When using an eZ DFS File Handler please refer to the settings in file.ini [eZDFSClusteringSettings].

Examples

DBPort=3306

By default this setting is set to port 3306.
