LDAPServer

Summary

Sets the LDAPhost

Usage

 
LDAPServer=<LDAP host>

Description

This setting sets the LDAP host. If the setting LDAPEnabled is set to true, the LDAP server set here will be used as a repository for user and group information.

Values can be the name of the LDAP host or its IP-address.

This setting is case sensitive.

Examples

 

LDAPServer=ldaphost


Instead of using the name of the LDAP server, as used here, the IP address could also be used.
