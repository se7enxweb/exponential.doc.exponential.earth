FileFieldSeparator

Summary

Sets the separator.

Usage

FileFieldSeparator=<string separator>

Description

This setting sets the separator to use between the fields of a file name. Possible values include "tab", ";", ",", "\" and all other string separators.

Examples

Example 1

FileFieldSeparator=tab

This is the default setting. A tab will be used to separate the file fields.
Example 2

FileFieldSeparator=;

A ";" will be used to separate the file fields.
