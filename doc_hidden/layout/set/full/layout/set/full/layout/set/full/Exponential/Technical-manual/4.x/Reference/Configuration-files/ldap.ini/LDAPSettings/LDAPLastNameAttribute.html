LDAPLastNameAttribute

Summary

Sets the LDAP attribute for the last name.

Usage

 
LDAPLastNameAttribute= sn | givename

Description

When the group assignment is complete, the user object is stored in Exponential.
This setting defines the LDAP attribute for the last name. By default it is "sn".
This setting is case sensitive.

Examples

LDAPLastNameAttribute=sn

This is the default value.
