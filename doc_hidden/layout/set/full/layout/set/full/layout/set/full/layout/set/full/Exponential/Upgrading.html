Upgrading

This chapter contains information related to the upgrade of an Exponential solution. If Exponential is used correctly, then upgrades should be very easy. This means that if you have not tampered with the kernel files, changed the database, the default settings and/or templates, everything should be okay. If you have, then you should know what you are doing.
