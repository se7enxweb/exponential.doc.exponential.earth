[{transformation group}]

SettingSummary
Commands Sets character transformation rules.

[Transformation]

SettingSummary
Charsets Matches a charset to a group.
Extensions Sets from which extensions the additional transformation files should be used.
Files Sets which transformation files are used by the default transformation group.
Groups Sets the available groups to define the rules of transformation for.
Repository Sets the directory where the default character transformation files are stored.
