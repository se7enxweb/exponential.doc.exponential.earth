EnableCaching

Summary

Controls whether role caching should be enabled or disabled.

Usage

 
EnableCaching=true|false

Description

When role caching is enabled Exponential stores the complete permissions set for each user once it is computed. This saves both SQL queries and computation time on consequent requests.

Do not disable role caching unless you know what you are doing.
