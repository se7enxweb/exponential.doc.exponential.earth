HostMatchRegexpItem

Summary

Sets which submatch to use to determine the siteaccess name with regexp hostmatching.

Usage

 
HostMatchRegexpItem= number

Description

The setting HostMatchRegexp sets the regular expression that is used for matching.

See HostMatchType for examples and more information about host matching.
