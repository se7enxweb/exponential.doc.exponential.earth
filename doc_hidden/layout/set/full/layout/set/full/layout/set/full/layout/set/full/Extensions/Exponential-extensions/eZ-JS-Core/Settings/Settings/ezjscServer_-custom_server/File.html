File

Summary

Sets the file to use.

Usage

File=<location>

Description

This is an optional setting that sets which files to use. If not defined autoload system will be used on class name.

This setting is case sensitive.

Examples

Example 1

File=extension/ezjscore/classes/mycustomjsgenerator.php

Example 2 

[ezjscServer_ezjsc]
File=extension/ezjscore/classes/ezjscserverfunctionsjs.php
 
[ezjscServer_ezjscnode]
File=extension/ezjscore/classes/ezjscserverfunctionsnode.php

In the above example it is shown how different files can be set for different custom servers.
