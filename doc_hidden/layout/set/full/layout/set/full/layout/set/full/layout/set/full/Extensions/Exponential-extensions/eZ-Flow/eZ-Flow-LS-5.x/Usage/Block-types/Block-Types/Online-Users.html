Online Users

An Online User Block can be added to displays the number of users that are online. This block is an Ajax based feature and the reloading of the online users is done using an Ajax call.

Log in to the eZ Flow front-end editing interface to edit your front page and select a zone for your Online Users block. In the block type list select Online Users.
