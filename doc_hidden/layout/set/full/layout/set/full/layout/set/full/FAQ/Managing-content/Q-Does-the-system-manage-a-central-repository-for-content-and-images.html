Q: Does the system manage a central repository for content and images?

A: Images (and all other binary elements) are stored as any other content in the system with a different versions created whenever the content changes. All images are stored in the storage directory defined (either on the filesystem or in the database, configurable). Images are permission controlled as any other content in the system.
