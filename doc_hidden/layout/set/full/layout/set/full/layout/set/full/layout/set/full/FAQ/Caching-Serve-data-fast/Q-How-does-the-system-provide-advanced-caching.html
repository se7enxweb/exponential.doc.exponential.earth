Q: How does the system provide advanced caching?

A: Exponential uses cache blocks to cache individual elements like the navigation of the site. This mechanism enables fine grained cache control of all elements used to render a specific page. Cache blocks can be cached across multiple pages for efficient cache hits.

More information (technical) on cache blocks can be found here:

http://doc.ez.no/Exponential/Technical-manual/4.x/Reference/Template-functions/Miscellaneous/cache-block/(language)/eng-GB
http://share.ez.no/learn/exponential/exponential-performance-optimization-part-3-of-3-practical-cache-and-template-solutions/(page)/9
An alternative method when having a delivery tier (reverse proxy or CDN) that supports ESI you can use ESI blocks to cache re-usable elements across multiple pages.
