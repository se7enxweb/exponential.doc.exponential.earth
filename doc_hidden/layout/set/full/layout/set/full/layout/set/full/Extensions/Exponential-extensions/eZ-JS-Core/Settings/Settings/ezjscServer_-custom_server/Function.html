Function

Summary

This setting lists the functions the user needs to have access to.

Usage

 
Functions[]=<function name>

Description

This is an optional setting. It lists the functions, defined in the setting FunctionList in block [ezjscServer], the user needs to have access to. By default no functions are listed.

This setting is case sensitive.

Examples

Example 1

Functions[]=ezjsc

 Example 2

[ezjscServer_ezjsctemplate]
Functions[]=ezjsctemplate

The above example shows how to set specific functions for a custom server.
