EditDirtyObjectAction

Summary

Sets what Exponential should do if the user tries to edit a page that has a draft that is newer than the published version.

Usage

 
EditDirtyObjectAction=showversions|usecurrent

Description

If you select showversions Exponential will display a page with all the available versions, including drafts. The user can then select which version he wants to base the new draft on.
If you select usecurrent Exponential will create a new draft based on the published version and let the user edit that.
