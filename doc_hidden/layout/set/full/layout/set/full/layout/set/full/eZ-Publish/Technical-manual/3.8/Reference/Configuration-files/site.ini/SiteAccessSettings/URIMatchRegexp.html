URIMatchRegexp

Summary

Sets the regular expression that is used to extract the siteaccess from the URI.

Usage

 
URIMatchRegexp= regular_expression

Description

Use the setting URIMatchRegexpItem to determine the submatch that contains the siteaccess.

See URIMatchType for examples and more information about URI matching.
