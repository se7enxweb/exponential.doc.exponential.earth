[TreeMenu]

SettingSummary
Dynamic Sets which implementation of the tree menu that should be used.
MaxDepth Sets the maximal depth for the tree menu.
RootNodeID Sets the root node for the tree menu.
ShowClasses Sets which types of nodes to show in the tree menu.
