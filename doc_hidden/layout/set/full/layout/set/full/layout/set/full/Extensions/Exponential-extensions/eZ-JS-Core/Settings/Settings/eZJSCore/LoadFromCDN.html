LoadFromCDN

Summary

Setting to enable or disable the ability to load files from external servers.

Usage

LoadFromCDN=enabled | disabled

Description

This setting will enable or disable the ability to load files from external servers.Additionally, when enabled the setting ExternalScripts in the same block is used for server location. When disabled the setting LocalScripts is used for server location. When yui is used, the setting LocalScriptBasePath will be used.

This setting is enabled by default.

This setting is case sensitive.

Examples

Example
[ezjscore]
LoadFromCDN=enabled
ExternalScripts[yui3]=http://yui.yahooapis.com/3.1.1/build/yui/yui-min.js

The system is allowed to load from external servers. The location of this server is defined with the setting ExternalScripts.
