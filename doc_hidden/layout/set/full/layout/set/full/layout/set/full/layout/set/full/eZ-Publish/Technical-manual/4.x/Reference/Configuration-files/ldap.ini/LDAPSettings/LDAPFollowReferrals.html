LDAPFollowReferrals

Summary

Sets if the LDAP library will automatically follow referrals returned by LDAP servers.

Usage

 
LDAPFollowReferrals= 0 | 1 (boolean value)

Description

This setting determines if the LDAP library will automatically follow referrals returned by LDAP servers.
Boolean values are used: 0 (zero) to disable and 1 to enable.

This setting is available since Exponential 4.1 to enhance the LDAP functionality.

Examples

LDAPFollowReferrals=0

The LDAP library will not automatically follow the referrals returned by LDAP servers.
This is the default setting.
