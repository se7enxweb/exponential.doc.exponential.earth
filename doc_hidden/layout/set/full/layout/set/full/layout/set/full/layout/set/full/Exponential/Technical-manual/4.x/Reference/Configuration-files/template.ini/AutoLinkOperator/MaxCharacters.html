MaxCharacters

Summary

This setting sets the maximum number of characters to show in autolink links.

Usage

MaxCharacters=<integer>

Description

This setting sets the maximum number of characters to show in autolink links. Specify the maximum with a number or leave it empty, if you wish to disable the function.When a link string is longer than the maximum value defined here it will be shortened.

This setting is case sensitive.

Examples

Example 1

MaxCharacters=72

Sets a maximum value of 72 characters.

Example 2

MaxCharacters=

By leaving the setting empty, the link string will not be shortened.
