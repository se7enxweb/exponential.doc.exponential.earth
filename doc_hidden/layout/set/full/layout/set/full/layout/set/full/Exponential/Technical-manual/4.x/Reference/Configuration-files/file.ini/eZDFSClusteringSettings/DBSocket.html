DBSocket

Summary

Enables the use of sockets by eZ DFS FIle Handler.

Usage

DBSocket=

Description

When using eZDFS File Handler it isn't possible to use a socket, for this reason "DBSocket" needs to be left empty and any other value will cause a fatal error of type "unable to connect to database...".

Examples

DBSocket=False
