Scripts

Exponential comes with a collection of scripts that cover the needs of various typical tasks. The scripts are grouped based on their type and purpose. The following table contains an overview of available script groups.

NameSummary
Generic Generic scripts for manual maintenance.
Upgrade Scripts for updating the existing content of sites when upgrading from older versions.
Cronjobs Scripts for automated periodic and scheduled maintenance.
Miscellaneous Scripts that do not belong to any of the groups above.
