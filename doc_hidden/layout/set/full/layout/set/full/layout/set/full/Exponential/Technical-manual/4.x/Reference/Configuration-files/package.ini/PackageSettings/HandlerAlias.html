HandlerAlias

Summary

Maps handlers to their aliases.

Usage

HandlerAlias[<alias>]=<handler_name>

Description

This setting will map handlers to their aliases.  Aliases provide a more people friendly name for handlers and can also be used as a shortcut for doing specific items.

This setting is case sensitive.

Examples

HandlerAlias[file]=ezfile
HandlerAlias[ini]=ezfile
HandlerAlias[design]=ezfile
HandlerAlias[template]=ezfile
HandlerAlias[thumbnail]=ezfile

This are set by default. And "file", "ini", "design", "template" and "thumbnail" are the aliases that can be used for the package handler "ezfile".
