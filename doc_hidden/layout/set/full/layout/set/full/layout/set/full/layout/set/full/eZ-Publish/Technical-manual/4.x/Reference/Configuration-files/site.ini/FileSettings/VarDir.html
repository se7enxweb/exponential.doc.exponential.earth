VarDir

Summary

Sets the main directory for file storage in Exponential.

Usage

VarDir=relative_path_to_directory

Description

The directory specified by this setting is used by Exponential for file storage. This includes content related files, cache files and other temporary files.

The value of this setting must be the same for all related siteaccesses (those sharing the same database). This is needed for the view cache clearing subsystem to function properly. (Otherwise, it won't clear the view cache for related siteaccesses when it is cleared for the current one.)

Similarly, do not use the same VarDir setting for siteaccesses that use different databases.

Note1: Do not change the default value unless you know what you are doing.

Note2: Only directories relative to the eZ Publish root are supported. Absolute directories are NOT SUPPORTED. Bear in mind that using an absolute directory will be accepted, but it won't work with some features and is NOT SUPPORTED.

Examples

VarDir=var

The directory "var/" is used to store files.
