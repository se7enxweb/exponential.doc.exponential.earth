TagPresets

Summary

setting to define sets of tags

Usage

TagPresets[]=

Description

This setting is available since Exponential 4.1. It enables you to define different sets of tags for different levels or types of content editors when using the Online Editor. For example, you might have a full set of tags for an administrator but a subset of tags for specific editors. Note that this is only used by Online Editor 5.0 to limit visible buttons.

Examples

Example 1

TagPresets[full]=All tags
