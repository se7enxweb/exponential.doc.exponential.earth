GeneratePasswordIfEmpty

Summary

Sets if Exponential should generate passwords if the password field is empty.

Usage

 
GeneratePasswordIfEmpty=true|false

Description

If GeneratePasswordIfEmpty is set to true, Exponential will automatically accept and generate passwords for users if the password field is empty. You can simply leave out the password field in the user register template.

If GeneratePasswordIfEmpty is set to false the user must select a password that is at least three characters long. Shorter passwords will not be accepted.
