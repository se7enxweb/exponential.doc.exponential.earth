Family

Summary

Sets a specific font family for the style defined by the configuration block.

Usage

Family=<family_name>

Description

This setting sets a specific font family for the style defined by the configuration block.

The values used must have a related .ttf file in the directory defined by the setting FontDir in [PathSettings], for example archtura.ttf.

This setting is case sensitive.

Examples

[gallery]
 Family=bluehigc
 PointSize=18
 XAdjustment=0
 YAdjustment=0
 WidthAdjustment=8
 HeightAdjustment=6
 BackgroundColor=#ffffff
 TextColor=#000000

These settings define the specific settings for the "gallery" style. Fonts are used according to the specifications defined in bluehigc.tff (location of this file is set with FontDir in [PathSettings]). Although the fonts from the family "bluehigc" are used, the point size, width, height and colors are adjusted according to the settings here.
