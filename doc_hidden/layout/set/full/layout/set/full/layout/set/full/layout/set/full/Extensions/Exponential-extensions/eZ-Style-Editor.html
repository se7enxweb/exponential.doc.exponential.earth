eZ Style Editor

The Exponential Style Editor is a JavaScript based application written on top of Yahoo! User Interface Library (YUI in short), as well as using power of ezjscore, the Exponential Ajax framework. Amongst other things, it provides a set of features which allow visual style editing for Exponential based extensions such as Website Interface and eZ Flow. It can be plugged to almost every existing Exponential installation and configured for custom styles and various DOM structures. It allows you to change the overall look and feel of Exponential based websites in visual edit mode.

Here you will find the documentation regarding the Exponential Style Editor solution. It covers the requirements, the installation procedure, the configuration, as well as the functional part of this solution, the styling.
