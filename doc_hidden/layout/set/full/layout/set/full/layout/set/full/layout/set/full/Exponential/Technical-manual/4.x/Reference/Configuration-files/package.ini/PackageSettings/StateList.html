StateList

Summary

This setting lists the possible states of the packages

Usage

StateList[]=

Description

This setting lists the state of packages.

This setting is case sensitive.

Examples

StateList[]
 StateList[mature]=Mature
 StateList[stable]=Stable
 StateList[beta]=Beta
 StateList[alpha]=Alpha
 StateList[prealpha]=Pre-Alpha

These are the default settings.
