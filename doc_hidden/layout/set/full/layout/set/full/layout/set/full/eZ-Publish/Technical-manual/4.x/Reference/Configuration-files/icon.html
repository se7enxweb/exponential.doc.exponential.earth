icon.ini

[FlagIcons]

SettingSummary
DefaultIcon Sets the default icon to use.
IconFormat Sets which suffix to use for flag icon files.
Repository Sets the location of the Flag icons repository.
Theme Sets which theme to use for the flag icons.

[IconSettings]

SettingSummary
Repsitory Sets the default location of the icon repository.
Size Sets the default icon size.
Sizes Sets override and new sizes.
Theme Sets the default theme to use for all icon types.

[{type}Icons]

SettingSummary
Size Sets which size to use for the icon type.
Theme Sets which theme to use for the icon type.
{type}Map Sets which map to use for the icon type.
