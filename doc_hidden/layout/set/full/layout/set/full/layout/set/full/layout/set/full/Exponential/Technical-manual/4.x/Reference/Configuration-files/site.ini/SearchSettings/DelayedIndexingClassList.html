DelayedIndexingClassList

Summary

Sets the class identifiers that should be used to delay indexing with.

Usage

DelayedIndexingClassList[]=

Description

This setting is new since Exponential 4.1. It contains a list of class identifiers that should be used to delay indexing with.

The value setted in this list is valid only if you have set the DelayedIndexing to the classbased value, otherwise this option is ignored.
