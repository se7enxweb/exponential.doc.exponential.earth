Q: How are cascading styles-sheets (CSS) supported?

A: There are two ways of supporting CSS in Exponential.

  • The most common is manual creation of the CSS. However this does require knowledge about CSS and XHTML.
  • The other option is to use the Style Editor in Exponential that enables you to simply modify the CSS (or parts that you are allowed to modify) for simple styling of colors, images, fonts etc. This enables efficient roll-out of sites with custom layout, without the need for any CSS or XHTML knowledge.

Documentation on the Style Editor can be found here:

http://doc.ez.no/Extensions/Exponential-extensions/eZ-Style-Editor/Styling
