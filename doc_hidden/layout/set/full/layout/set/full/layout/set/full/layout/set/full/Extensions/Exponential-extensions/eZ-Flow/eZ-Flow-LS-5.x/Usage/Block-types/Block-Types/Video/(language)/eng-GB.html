Video

This block is designed to ensure an easy an optimal video display. To give you an idea of how this looks on the front page, a video block is marked in red in the following screenshot.

 

To add such a block, log in to the eZ Flow front-end editing interface and edit the front page. Select the correct zone and select "Video (manual)" in the block type drop down list.

Select "Add to block". The following block will be added to the list. Here you can name the block, and add the video's which are meant to be shown. It is also possible to set the overflow of this block, and the rotation frequency of the images, set in minutes, hours or days.

Click "Send for publishing" to complete the procedure.

Note:

It can be useful to embed a video in an article on your website and with eZ Flow this can be done with the video custom tag. Please visit the following page for more information.
