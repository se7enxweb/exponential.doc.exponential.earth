Q: Are different language letter accenting & reading styles catered for?

A: Exponential supports the different character sets by being fully Unicode. All character sets defined above is fully supported.
The different reading style is managed by Unicode (content) and templates (labels, navigation etc). You have full control over defining the desired reading style for any given content.
