AllowedFilters

Summary

Sets the allowed filters.

Usage

 
AllowedFilters[]=<filter>

Description

A list of filters which are allowed to be run on this image, wildcards can also be used.If this is supplied only filters that are in this list will be allowed.Note that resizing of animated GIFS is not supported when using GD2, at least not until PHP 5.3.

For more information please visit: http://www.libgd.org/FAQ_PHP

This setting is case sensitive.

Examples

AllowedFilters[]=*
