Charset

Summary

Sets the character set that Exponential should use

Usage

 
Charset=character_set

Description

Use this directive to set the internal charset for the site. Specify UTF-8 for multilingual sites or a specific charset for monolingual sites.

Examples

Example 1

Charset=utf-8

This will set the output character set of Exponential to UTF-8.
