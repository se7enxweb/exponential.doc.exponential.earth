ClearCacheBlocks

Summary

This is a new optional setting for clearing cache blocks available in version 4.3 and upwards of Exponential

Usage

 
ClearCacheBlocks=disabled

Description

All cache blocks without ignore_content_expiry are cleared on publish, with this you can disable it for selected classes.

The default setting is:

#ClearCacheBlocks=enabled
