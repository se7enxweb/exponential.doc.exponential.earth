ExtensionDirectories

Summary

Lists the extensions which support package handlers.

Usage

ExtensionDirectories[]=<name_extension>

Description

This setting lists the extensions which support package handlers. For these extensions the package handlers must be placed in the subdirectory "packagehandlers" located here:...(root of your Exponential installation)/extension/<EXTENSION_NAME>/packagehandlers

By default this setting is left empty.

This setting is case sensitive and left empty by default.
