Extensions

Summary

Sets the PHP required session extensions.

Usage

 
Extensions=<name PHP extension>

Description

Some PHP extensions are required for Exponential to work correctly. During setup it is tested if these extensions exist. This setting lists the extensions that need to be tested. The setting "Require" in the same block defines whether only one of these extensions must exist, or whether all of them should.
This setting is case sensitive.

Examples

Extensions=session

"session" is the default value for this setting.
