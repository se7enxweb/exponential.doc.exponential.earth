MultivariateTestingHandlerClass

Summary

A handler class which implements <ezpMultivariateTestHandlerInterface> interface

Description

Optionally you can define your custom handler class (more information how to create your own handler class in the “Create custom handler” section of ezmultivariatetest doc bundled in the extension). This step can be skipped, then default implementation will be used.

Examples

# A handler class which implements <ezpMultivariateTestHandlerInterface> interface
MultivariateTestingHandlerClass=ezpMultivariateTestHandler
