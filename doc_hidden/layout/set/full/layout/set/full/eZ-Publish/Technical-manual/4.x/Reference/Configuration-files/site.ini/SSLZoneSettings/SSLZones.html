SSLZones

Summary

Enable/disable the SSL zones functionality.

Usage

SSLZones=enabled|disabled

Description

This setting allows enabling or disabling the SSL zones functionality. It expects enabled or disabled as value.

With SSL Zones enabled you can define the subtrees where SSL will be used. For that, the subtrees must be defined using the SSLSubtrees setting.

You can also define the view access mode for each module. For more details please check the ModuleViewAccessMode setting.

Examples

SSLZones=enabled
