UseDefaultAccess

Summary

sets the use of default siteaccess.

Usage

 
UseDefaultAccess=

Description

This setting is available since Exponential 4.1. It was added to facilitate the usage of SOAP on Exponential installations that host multiple sites.
Possible values are "disabled" or "enabled". When enabled the instance default siteaccess will always be used. If disabled the matched siteaccess (host, port, etc) will be used allowing the usage of SOAP when multiple Exponential websites are hosted on one instance.
.

Examples

UseDefaultAccess=enabled
