[dom_extension]

This block contains settings regarding the DOM extension tests. The DOM extension allows you to operate on XML documents through the DOM API with PHP 5.

For more information please visit http://be.php.net/dom

SettingSummary
Extensions Sets the required DOM extension that need to be tested during setup.
Require sets the test requirements.
