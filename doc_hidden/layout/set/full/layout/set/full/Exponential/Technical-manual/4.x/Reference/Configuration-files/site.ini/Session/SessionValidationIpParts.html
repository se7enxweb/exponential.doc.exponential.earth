SessionValidationIpParts

Summary

Sets how many parts of the clients IP must be used for session validation

Usage

 
SessionValidationIpParts=

Description

This setting is available since Exponential 4.1. It is an optional setting which increases session security by enabling session validation by defining how many parts of the clients IP must be used for session validation. Possible values are 0 to 4.

Note that this setting can't be used if load balancing is used for the connection or if the site is behind reverse proxy.

Examples

SessionValidationIpParts=0
