SendmailOptions

Summary

Sets the additional sendmail options.

Usage

 

SendmailOptions[]= option1 

SendmailOptions[]= option2 
...

Examples

SendmailOptions[]=-r
SendmailOptions[]=nospam@ez.no

The resulting option "-r nospam@ez.no" will be passed to sendmail.
