ExtensionRepositories

Summary

Lists the extensions which have modules.

Usage

 
ExtensionRepositories[]=<name_extension>

Description

This setting lists the extensions which have modules.
It is recommended to create a module.ini.append.php file within these extensions, at the following location:
(...root of your Exponential installation)/extension/<extension>/settings/module.ini.append
By adding the extension name to that file, the modules from the extension will automatically be loaded when the extension is turned on.
This setting is case sensitive.
By default this setting is left empty.
