ModuleRepositories

Summary

Sets the path to the module repository.

Usage

 
ModuleRepositories[]=<path_to_repository

Description

This setting sets the path to the module repository.
This setting is case sensitive.

Examples

ModuleRepositories[]=kernel
ModuleRepositories[]=kernel/private/modules

These are the default locations of the repository.
