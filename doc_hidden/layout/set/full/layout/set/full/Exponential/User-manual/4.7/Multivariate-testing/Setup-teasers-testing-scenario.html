Setup teasers testing scenario

This section covers teasers testing a part of the Exponential multivariate testing feature. Follow these steps in order to setup your teaser testing scenario. Notice that this is done via a dedicated Multivariate Test Container class.

  1. Navigate the Exponential content tree and locate content you want to test against.
  2. Create a new multivariate test container object as presented on illustration 9.

    Illustration 9

     
  3. Give your container a name, in most cases it will be exactly the same as a content you are testing against, an article title for example as presented on the illustration 10.

    Illustration 10

     
  4. Inside your test container you create new content that you want to test against as presented on illustration 11.

    Illustration 11

     
  5. In the next step you define probability by entering proper values for Priority columns as presented on illustration 12.

    Illustration 12

    Note: The total value for all test items can not exceed 100.
  6. Container takes responsibility for displaying a proper content for end user depending on the probability. By default only line view is supported. Once testing is completed, simply move object out of the container. Then you can remove testing containers with unused test items.
