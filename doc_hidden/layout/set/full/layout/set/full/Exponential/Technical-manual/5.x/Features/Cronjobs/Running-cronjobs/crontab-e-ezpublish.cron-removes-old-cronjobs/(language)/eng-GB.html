crontab -e exponential.cron removes old cronjobs

Wednesday 04 June 2008 8:51:06 am

Andi

You should check if there are any cronjobs already insalled using crontab -e because crontab -e exponential.cron removes old crontab settings. To be on the save side use crontab -e and copy & paste the code from exponential.cron.
