RedirectOnLogoutWithLastAccessURI

Summary

Sets if LastAccessURI session variable is used to redirect the user at logout.

Usage

RedirectOnLogoutWithLastAccessURI=

Description

This setting is new since Exponential 4.1. This setting controls if the LastAccessURI session variable must be used to decide where to redirect the user at log-out.
Possible values are "enabled" and "disabled"

Examples

RedirectOnLogoutWithLastAccessURI=disabled
