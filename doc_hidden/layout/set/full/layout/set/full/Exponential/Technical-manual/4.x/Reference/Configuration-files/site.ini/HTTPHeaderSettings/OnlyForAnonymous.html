OnlyForAnonymous

Summary

Setting to set custom HeaderList for anonymous users only

Usage

OnlyForAnonymous=enabled

Description

This setting is available since Exponential 4.1. It allows you to specify that custom HeaderList are only intended for anonymous users.

It can be enabled or disabled.

Examples

Default:

OnlyForAnonymous=disabled
