Conclusion

With a simple example we have learned several techniques useful for creating Exponential extensions.

In addition to creating custom modules with different views and view parameters, template fetch functions and template operators, we also looked at the privilege system of Exponential and now know how to write custom messages in the Debug view or in Log files. We also looked at accessing INI files.

With this basic knowledge it should be possible for you to create your own Exponential extensions. You can download the source code for the tutorial at: http://projects.ez.no/jacextension
