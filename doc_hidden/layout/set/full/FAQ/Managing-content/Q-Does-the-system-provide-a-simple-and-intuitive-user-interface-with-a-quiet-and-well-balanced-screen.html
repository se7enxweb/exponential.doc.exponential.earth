Q: Does the system provide a simple and intuitive user interface with a quiet and well-balanced screen?

A: Exponential comes with a WYSIWYG editor.

You can insert externally formatted HTML into the online editor. However since Exponential is strictly based on validated XML there are two modes this can be done under. The first is to have Exponential wash the HTML and convert it to valid XML, the other is to use literal definitions that enables the editor to input the actual HTML that will be rendered. The latter is useful if you need to insert some very specific HTML like for example an embedded YouTube video code.

All literal definitions can automatically be filtered away if the content is delivered to other channels (e.g. for mobile Apps or print where the HTML would not work).
