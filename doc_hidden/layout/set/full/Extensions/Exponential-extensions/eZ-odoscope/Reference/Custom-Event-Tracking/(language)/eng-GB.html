Custom Event Tracking

To track any kind of events, you can use the "osc.evt(eventName, eventValue)" javascript function.

Here is a simple example:

onClick=osc.evt("Download","faq.pdf")

This will tell odoscope to create a filter group “Download” with a filter “faq.pdf”. That way, it is possible to identify all of the website visitors who downloaded the faq.

You can also add hierarchy information to your filters. An example: If you have two regions where different faqs can be downloaded you can separate these with an underscore like that:

onClick=osc.evt("Download","regionOne_faq.pdf")

onClick=osc.evt("Download","regionTwo_faq.pdf")

As some browsers or browser extensions may transfer data that could be interpreted as a new filter group, there is a white listing. So, if you setup a new filter group, please drop us a line and we will activate it for you.
