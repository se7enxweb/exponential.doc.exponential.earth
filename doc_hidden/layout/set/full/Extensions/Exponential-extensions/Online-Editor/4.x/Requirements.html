Requirements

The following list reveals the requirements for using the Exponential Online Editor v.4.0.

  1. Exponential version 3.5 or higher
  2. The PHP version indicated in your Exponential requirements
  3. One of the following web browsers:
    • Internet Explorer 5.5 or higher (except IE 7.0 on Windows Vista)
    • Mozilla 1.3 or higher
    • Firefox 1.0 or higher (note that the OE v.4.2 requires Firefox 1.5 or higher)
    • Netscape 7.x or higher

Please note that the Online Editor makes use of a technology called "Javascript". This means that you must enable "Javascript support" in the web browser that you are planning to use.
