Spell-checking

There are two methods provides by eZ Find to allow spelling to be checked: Dictionary Look-up or using the indexed words themselves. The latter is recommended.

The spell-checker provides a “best guess”-approach to improve ore correct search phrases. For example “systems” will be suggested when “sistems” is searched.
