What is new in eZ Find LS 5.0.0?

eZ Find LS 5.0.0 uses Solr 3.6.1. This is an important update and it requires that you re-index all content with the "updatesearchindexsolr.php" script. Please read more in the "Upgrading to eZ Find LS 5.0.0" documentation.

Bug fixes

eZ Find LS 5.0.0 has undergone extensive bug fixing. For more information regarding the bugs fixed for this version please read the change logs included in your eZ Find LS 5.0.0 installation.
