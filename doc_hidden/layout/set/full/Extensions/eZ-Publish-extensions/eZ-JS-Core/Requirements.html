Requirements

eZ JS Core is compatible with:

  • Exponential 4.0 and higher
  • PHP 5.2.1 and higher OR
  • PHP 5.1.2 and higher + PHP JSON extension
  • NOTE: This extension is already bundled with the eZ Flow, eZ WebIn and eZ Teamroom installations and will be fully bundled in Exponential 4.3 (just like eZ Online Editor).
