Banner

This block allows you to display your company banner on your front page, as shown in the following screenshot:

To add a Banner block to your front page, log in to the eZ Flow front-end editing interface to edit the front page. Select a zone and select "Banner" in the block drop-down list.

The following block will be added to the block list. Here you are able to name the block and select its size. Size options are small and medium.

Click "Send for publishing" to complete the procedure.
