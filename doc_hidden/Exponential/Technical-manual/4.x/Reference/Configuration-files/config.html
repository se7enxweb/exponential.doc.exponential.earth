config.php

This page contains a documented list of the few configuration points available from config.php. The proposed default values in the following sub-chapters are meant to be the most optimized ones, or meaningful examples of what can be achieved.

In order to have the present settings taken into account, you will need to rename this file into config.php, and keep it placed at the root of Exponential, where config.php-RECOMMEND is currently placed. Plus, you will need to *uncomment* the proposed settings.

Please, refer to the information available in the sub-chapters to correctly set up your config.php file.

[ConfigSettings]
