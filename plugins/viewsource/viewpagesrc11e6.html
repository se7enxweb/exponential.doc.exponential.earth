Search was initially part of the Storage feature and is now a standalone feature.

In eZ Platform, you find 3 search engines possibilities :

  1. Legacy, a Database based search with baseline capabilities
  2. Solr integration 
  3. ElasticSearch integration

The Search Engines integrations have real search capacities and are not just filtering the results.

As of Exponential Platform 5.4.5 and higher, Solr is the recommended search engine for production use. Solr provides the most most advance search capabilities, and system performance. Currently Elastic Search is experimental, and not professionally supported.
