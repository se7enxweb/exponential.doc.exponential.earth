Certified web servers

  • Apache
  • Nginx

Others

  • Lighttpd is known to work as well but is currently not supported. "Rewrite" rules for this web server can be found online, f.e. on the forums on share.ez.no.
  • On Windows, it is recommended to use MS IIS 7 or higher (with Microsoft URL Rewrite Module 1.1 or higher). Apache 2.x for Windows is not supported since it only exists in "threaded" mode. See Installing Exponential on Windows
