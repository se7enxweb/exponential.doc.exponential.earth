Documentation on how to use Exponential 5, how to develop on top of it.

There are two official approches to building applications on top of Exponential 5; REST API OR using Public API and MVC/Application layer (Symfony2 based).
In the latter case Public SPI provides the "model" (the data from the database and the file system) and the MVC/Application layer provides all the application logic (configuration, routing, controlling, ...).
