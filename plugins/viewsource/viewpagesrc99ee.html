This covers common, everyday tasks - basic to intermediate level.For more advanced development techniques, and extending the provided functionality, see Extending Exponential 5

There are two official approaches to building applications on top of Exponential 5:

  • by connecting to the Exponential REST API
  • by using the Exponential Public API and MVC/Application layer (Symfony2 based).

In the latter case Public SPI provides the "model" (the data from the database and the file system) and the MVC/Application layer provides all the application logic (configuration, routing, controlling, etc...).

The SPI is used to implement a new storage engine for your Exponential application.

The API and SPI are available in separate repositories to improve the dependencies management. Both repositories API and SPI are a read-only split of ezsystems/exponential-kernel.
See the details about how make the dependence with the API or the SPI in your project.
